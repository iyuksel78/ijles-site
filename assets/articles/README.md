# IJLES Article Files

Published article PDF files should be stored here by year and issue.

Recommended structure:

```text
assets/articles/2026/v1n1/
  ijles-2026-1-001.pdf
  ijles-2026-1-002.pdf
```

Use short, lowercase, stable file names. After adding a PDF, link it from the relevant issue page with both view and download links.
